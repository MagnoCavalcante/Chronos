# CHRONOS — Brand Colors Reference (Hex & RGBA)
## Versão 1.0 (Sprint 5.2.3)

Este arquivo especifica as fórmulas de cor oficiais do CHRONOS para fácil importação em implementações Sass, Tailwind, CSS nativo ou Flutter.

### 1. Primary Blue (Azul Profundo)
- **Hex**: `#1E3A5F`
- **RGB**: `rgb(30, 58, 95)`
- **HSL**: `hsl(214, 52%, 25%)`

### 2. Accent Gold (Dourado Antigo)
- **Hex**: `#C9A227`
- **RGB**: `rgb(201, 162, 39)`
- **HSL**: `hsl(45, 68%, 47%)`

### 3. Light Background (Canvas Off-white)
- **Hex**: `#F8F6F0`
- **RGB**: `rgb(248, 246, 240)`
- **HSL**: `hsl(45, 27%, 96%)`

### 4. Dark Charcoal Text
- **Hex**: `#1F2937`
- **RGB**: `rgb(31, 41, 55)`
- **HSL**: `hsl(215, 28%, 17%)`
