# CHRONOS — Typography Reference & CSS Styles
## Versão 1.0 (Sprint 5.2.3)

Este arquivo especifica as regras de tipografia para títulos, corpo de texto e metadados.

### 1. Font Families
- **Display Headings**: `font-family: 'Cinzel', Georgia, serif;`
- **Body & Controls**: `font-family: 'Inter', system-ui, sans-serif;`
- **Monospace Metadata**: `font-family: 'JetBrains Mono', monospace;`

### 2. Classes de Tamanho de Fonte (Tailwind Equivalence)
- **H1 (Large Titles)**: `text-4xl font-semibold tracking-tight` (36px)
- **H2 (Page Headers)**: `text-2xl font-medium tracking-tight` (24px)
- **H3 (Section Title)**: `text-lg font-medium tracking-normal` (18px)
- **Body Large**: `text-base font-normal leading-relaxed` (16px)
- **Body Regular**: `text-sm font-normal leading-normal` (14px)
- **Metadata Small**: `text-xs font-medium leading-none` (12px)
