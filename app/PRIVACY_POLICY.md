# Política de Privacidade - CHRONOS

**Última atualização:** 23 de julho de 2026

## 1. Informações Coletadas

O CHRONOS coleta e armazena localmente no dispositivo do usuário:
- Progresso de estudos
- Favoritos e coleções
- Preferências de configuração
- Histórico de pesquisa local
- Métricas de uso (analytics local)

## 2. Dados NÃO Coletados

O CHRONOS **NÃO** coleta:
- Dados pessoais identificáveis (nome, email, telefone)
- Localização GPS
- Contatos
- Fotos ou arquivos pessoais
- Dados de navegação em outros aplicativos

## 3. Armazenamento

Todos os dados são armazenados:
- Localmente no dispositivo (SharedPreferences)
- No backend Supabase com criptografia (quando sincronizado)

## 4. Compartilhamento

O CHRONOS **NÃO** compartilha dados com terceiros.

Nenhuma informação é vendida, alugada ou cedida.

## 5. Analytics

O sistema de analytics é **100% local**:
- Nenhum dado é enviado para servidores externos
- Registra apenas contagens de uso para melhorar a experiência
- Pode ser desativado nas configurações

## 6. Serviços de Terceiros

- **Supabase**: Backend para dados históricos (sem dados pessoais)
- **OpenStreetMap**: Tiles de mapa (sem rastreamento)
- **Provedores de IA** (opcional): Consultas de texto (sem dados pessoais)

## 7. Segurança

- Comunicação via HTTPS/TLS
- Autenticação via tokens JWT
- Row Level Security no banco de dados
- Nenhum dado sensível em texto plano

## 8. Direitos do Usuário

O usuário pode a qualquer momento:
- Limpar todos os dados locais
- Desativar analytics
- Desativar sincronização
- Excluir conta (quando aplicável)

## 9. Menores de Idade

O CHRONOS é um aplicativo educacional adequado para todas as idades.
Não coleta dados de menores de forma diferenciada.

## 10. Contato

Para dúvidas sobre privacidade:
- Email: privacy@chronos.app

## 11. Alterações

Esta política pode ser atualizada. Alterações significativas serão notificadas via tela "Novidades da Versão".
